What Are Event Listeners?

Event listeners in JavaScript allow the programmer to call a function when a certain event occurs on an HTML Element.

What Are Some Events for Event Listeners?

  • Click
  • Mouse Over
  • Key Pressed Down

...and more

When Are Event Listeners Useful?

The most common example of event listeners is a button. You can code buttons to do anything such as...

  • Hide/Show Elements
  • Display Messages
